# SMENA Construction Graphics 01

Единый графический набор для интерфейсов рабочего, бригадира и подрядчика.

## Форматы

- `svg/` — исходные масштабируемые активы с прозрачным холстом.
- `png/` — PNG 2x с настоящим alpha-каналом.
- `backgrounds/` — оптимизированные непрозрачные WebP-фоны карточек объектов.
- `graphics-manifest.json` — перечень файлов и базовая палитра.
- `transparency-report.json` — машинная проверка прозрачности каждого PNG.

Шахматная подложка существует только в экранной галерее прототипа и не входит ни в один графический файл.
Фоны карточек намеренно непрозрачны: это полноформатные 3D-сцены, которые перекрываются интерфейсным градиентом для читаемости текста.

## Повторный экспорт

```bash
npm run graphics:export
```
